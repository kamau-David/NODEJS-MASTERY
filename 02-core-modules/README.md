# 02 - Core Modules

## Concepts covered
- `fs` (file system) - sync vs async vs promises API
- `path` - cross-platform path handling
- `os` - system info
- `events` - the EventEmitter pattern (foundation of most of Node's async APIs)
- `process` - env vars, args, exit codes

## Files
- `fs-demo.js`
- `path-demo.js`
- `os-demo.js`
- `events-demo.js`
- `process-demo.js`

## Run it
```bash
node fs-demo.js
node path-demo.js
node os-demo.js
node events-demo.js
node process-demo.js hello world
```
