# 10 - Real-time with WebSockets (Socket.io)

## Concepts covered
- Why HTTP request/response doesn't work for real-time (client must poll)
- WebSockets: persistent, bi-directional connections
- Socket.io: rooms, broadcasting, event-based messaging
- Building a minimal live chat

## Files
- `server.js`
- `public/index.html` — simple chat client
- `package.json`

## Run it
```bash
npm install
node server.js
# open http://localhost:3005 in two browser tabs and chat between them
```
