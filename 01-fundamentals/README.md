# 01 - Fundamentals

## Concepts covered
- Variables & scope (`var` vs `let`/`const`)
- Functions (declarations, arrow functions, default params)
- CommonJS modules (`require`/`module.exports`) vs ESM (`import`/`export`)
- npm basics: `package.json`, installing packages, scripts

## Files
- `variables-scope.js` — scope pitfalls that trip up beginners
- `functions.js` — function styles and when to use each
- `math-utils.js` + `commonjs-demo.js` — CommonJS module pattern
- `esm-demo.mjs` — same thing, ESM style
- `package.json` — a real package.json with useful scripts

## Run it
```bash
node variables-scope.js
node functions.js
node commonjs-demo.js
node esm-demo.mjs
```
